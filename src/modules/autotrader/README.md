## Функционал автотрейдинга, который потенциально перерастет в отдельного бота

**Его задачи**
- мониторить сделки в которых он состоит
- подтягивать стопы
- изменять сделку по входным данным
- устанавливать сделки
  - Поддержка нескольких тейкпрофитов
  - Поддержка стопов
  - Все рассчеты должны быть включены
- рассчитывать риски и потенциальный профит
- Если сделка закроется по ТП убрать СЛ
- Если сделки закроется по СЛ убрать ТП
- Если часть сделок ордеров не выставится (ТП/СД/Маркет) то сделает ретрай или уведомит
- Уведомит о любых изменениях в чат
- Смотрит закрытие сделок
